    DisK

    Digtal assets Due Dilligence, Transition and Transformation processes automation tool.

    description

    DisK allows for discovery and processing of knowledge, aggregates and analyses the massive data acquired, scanned or assessed to create the collective information base needed to feed the data needs of transformation management platforms and to be the foundation for ad-hoc queries and complex compound analysis.

    DisK is the middle piece of end-to-end services for Transition and Transformation (T&T). It can be used throughout various stages, from an initial due diligence to the final execution of the transformation.

    DisK integrates structured and unstructured data from heterogeneous sources and formats into analysable aggregates in a Hadoop environment.

    DisK processes, aggregates, analyses and enriches:

    • classical inventory data, connected to financial information, contracts, SLAs and KPIs;
    • information with reference to applications and workflows;
    • information to identify business critical applications and workflows;
    • communication and contact relations in the IT and application landscape;
    • qualitative and quantitative experience information from technical and business related operations of the IT and application landscape;
    • information with reference to criticality, software, hardware and application maintenance, lifecycle and release management.

    DisK establishes and maintains a logical syntax of the analysable data aggregates.

    DisK Enterprise search uses the logical syntax to allow seamless target searches and complex compound analysis like GAP and risk analysis, complexity assessments or history based effort predictions.

    Project length

    30 months

    Anatree role

    Technical consultancy, technical product ownership, big data architecture design, data integration, development, deployment, maintenance.

    Customer value

    Consolidate 20 Data Centers into 5; save 25% on IT and personnel cost starting in Year 3. Optimize consulting costs to 50% thanks to DisK compound analysis and enterprise search tools.

    SIVote

    Ballots counting automation tool

    Description

    With the traditional way of voting that uses pen and paper a large numbers of voting cards are created. To aid the validation and votes counting processes Anatree have provided an automated IT tool that makes it simple, fast and reliable.

    The solution allows for parrallel scanning the voting cards, performs high-precision image recognition and calculates the ballots outcome. It also provides an interface for calculation validation.
